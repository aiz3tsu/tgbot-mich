# Бот предложки для Telegram-канала "Мичуринск Сейчас"

## Что делает:
- Принимает текст, фото, видео от пользователей
- Отправляет их в Telegram ID администратора

## Как запустить:
1. Установи зависимости:
   pip install -r requirements.txt

2. В файле main.py:
   - Вставь токен бота
   - Укажи свой Telegram ID

3. Запусти:
   python main.py
